# Finbroker — product redesign case study

Клиентская презентационная страница с продуктовой аргументацией, дизайн-системой и встроенным интерактивным прототипом.

## Возможности прототипа

- переключение Demo / Real с заменой контента;
- состояния Real без счёта и после открытия счёта;
- loading-состояние основного CTA;
- рабочие разделы Главная, Счета, Торговля, Операции и Профиль;
- UI-kit с состояниями кнопок;
- прямые ссылки на Figma и GitHub.

## Локальный запуск

```bash
python3 -m http.server 4173
```

Открыть: http://localhost:4173
